Just a simple to do list app with laravel and mysql as a database.

To get this app
- `git clone https://github.com/your-username/todo-list-app.git`
- `cd todo-list-app`
- `composer install`
- `cp .env.example .env`
- `php artisan key:generate`
- `php artisan migrate`
- `php artisan serve`
